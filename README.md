# Tic-Tac-Toe

A modern, responsive Tic-Tac-Toe game built with HTML, CSS (Tailwind CSS), and JavaScript.

## 🚀 Live Demo

Check out the live game here: [https://bg-tic-tac-toe.onrender.com](https://bg-tic-tac-toe.onrender.com)

## ✨ Features

- **Interactive Gameplay:** Play Tic-Tac-Toe against a friend on the same device.
- **Modern UI:** Sleek and premium design using Tailwind CSS.
- **Dynamic Animations:** Smooth transitions and hover effects for an enhanced user experience.
- **Winner Detection:** Automatically detects and displays the winner or a draw.
- **Game Reset:** Easily restart the game at any time with a single click.

## 🛠️ Built With

- **HTML5:** For the structural foundation.
- **Tailwind CSS:** For modern, responsive styling and animations.
- **JavaScript (ES6):** For game logic and interactive elements.

## 🚀 Getting Started

To run this project locally, follow these simple steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/bhavit-gupta/Tic-Tac-Toe.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd Tic-Tac-Toe
    ```
3.  **Open `src/index.html` in your browser.**

Alternatively, if you want to modify the Tailwind styles:

1.  **Install dependencies:**
    ```bash
    npm install
    ```
2.  **Run build:**
    ```bash
    npm run build
    ```
3.  **Run Tailwind in watch mode:**
    ```bash
    npm run watch
    ```
